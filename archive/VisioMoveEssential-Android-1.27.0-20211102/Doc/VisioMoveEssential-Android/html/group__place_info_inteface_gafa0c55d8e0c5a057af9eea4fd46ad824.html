◆ queryPlaces()

void queryPlaces ( VMEPlaceFilter  filter,
VMEPlaceFilterCallback  callback 
)

Queries the map for places matching the filter.

Parameters
filterThe place filter to be applied to the map.
callbackThe callback object to be notified of the results
Note
Applies to:
  • static places
  • dynamic places
private void queryPlacesExample()
{
VMEPlaceFilter lFilter = new VMEPlaceFilter("404");
lFilter.setRestrictToTargetLayer(false);
lFilter.setRadius((double) 50);
lFilter.setRestrictToPlaceIDs(Arrays.asList("411", "413", "206"));
mMapView.queryPlaces(lFilter, this);
}
@Override public void placeFilterDidFinish(VMEMapView mapView, VMEPlaceFilter filter, List<VMERouteResult> results)
{
for (VMERouteResult lResult : results) {
Log.i("INFO", lResult.getLength() + "m from " + lResult.getDestinations().get(0));
}
}
