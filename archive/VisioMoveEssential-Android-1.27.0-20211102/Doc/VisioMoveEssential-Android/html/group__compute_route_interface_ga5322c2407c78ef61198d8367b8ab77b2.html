◆ computeRoute()

void computeRoute ( VMERouteRequest  routeRequest,
VMEComputeRouteCallback  callback 
)

Requests that a route be computed.

Parameters
routeRequestThe route request
callbackThe callback to invoke with the result. Pass null if you do not need to be informed of the result.
Example
List<Object> lDests = new ArrayList<>();
lDests.addAll(Arrays.asList("B4-UL05-ID0032", "B2-LL01-ID0011", "B3-UL00-ID0070"));
Boolean lAccessible = false;
VMERouteRequest lRouteRequest = new VMERouteRequest(VMERouteRequestType.FASTEST, VMERouteDestinationsOrder.OPTIMAL, lAccessible);
lRouteRequest.setOrigin("B1-UL00-ID0039");
lRouteRequest.addDestinations(lDests);
mMapView.computeRoute(lRouteRequest, mRouteCallback);
